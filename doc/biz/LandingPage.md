# Tumiki LP（ランディングページ）構成案

## 1. ヒーローセクション
- **キャッチコピー**：「AIと外部ツールを瞬時に接続。業務自動化の新たな扉を開く」
- **サブキャッチ**：「複雑な開発は不要。クリックするだけでAIがSlack、Google、LINEなどあらゆるツールと連携。企業のAI活用を次のステージへ」
- **CTA（コールトゥアクション）ボタン**：「無料で始める」「3分でわかるデモ動画」
- **ヒーローイメージ**：AIと各種ツールが接続される様子のアニメーション

## 2. サービス概要
- **Tumikiとは何か**：「Tumikiは、AIと社内外のツールを簡単に接続できるクラウドプラットフォームです。従来は専門知識が必要だった外部ツール連携を、ノーコードで実現。AIによる業務自動化の可能性を大幅に拡張し、企業の生産性を飛躍的に向上させます」

## 3. 課題提起セクション
- **今までの課題**
  - AIを導入したが、社内ツールと連携できず活用範囲が限定的
  - 外部ツール連携のたびに高額な開発コストが発生
  - エンジニアがいないと新しい連携を追加できない
  - セキュリティが心配で外部連携に踏み切れない
- **Tumikiで実現できること**
  - ワンクリックで連携 - 100以上のツールと即座に接続
  - 開発不要 - エンジニアなしでAIの活用範囲を拡大
  - 安全な運用 - 企業レベルのセキュリティで安心
  - コスト削減 - 開発費用ゼロで新機能を追加

## 4. 活用シーン
- **マーケティング自動化**
  - LINE公式アカウント × AI：24時間顧客対応、個別最適化メッセージ
- **営業支援**
  - Salesforce × Slack × AI：商談情報通知、次のアクション提案
- **開発効率化**
  - GitHub × Jira × AI：自動コードレビュー、タスク自動生成
- **データ分析・レポート**
  - Google Sheets × BI ツール × AI：定期レポート自動生成、異常検知

## 5. 主な特徴・メリット
- **瞬時に接続**
  - 事前設定済みの接続テンプレート
  - 3ステップで連携完了
  - すぐに使える実用的なAIワークフロー
- **エンタープライズセキュリティ**
  - アクセス権限の細かな制御
  - 全ての操作ログを記録・監査
  - ISO27001準拠（取得予定）
- **柔軟なカスタマイズ**
  - AIの動作を自由に調整
  - 企業独自のワークフローを構築
  - APIで既存システムとも連携
