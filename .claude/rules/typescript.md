# TypeScript コーディングルール

| ルール         | 説明                                |
| -------------- | ----------------------------------- |
| `any`禁止      | `unknown`またはジェネリック型を使用 |
| `type`のみ     | `interface`禁止                     |
| アロー関数     | `function`禁止                      |
| 日本語コメント | コード内のコメントは日本語          |

**型インポート**: `@tumiki/db`から（`@prisma/client`禁止）
