# フロントエンド コーディングルール

## コンポーネント配置

| 場所                    | 用途       |
| ----------------------- | ---------- |
| `_components/`          | ページ専用 |
| 親階層の `_components/` | 共通利用   |
| `packages/ui/`          | アプリ横断 |
