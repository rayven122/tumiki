# 品質チェックコマンド

プロジェクトの品質チェックを順番に実行し、結果をサマリー表示します。

## 実行手順

1. **コードフォーマット修正**
   ```bash
   pnpm format:fix
   ```

2. **Lint修正**
   ```bash
   pnpm lint:fix
   ```

3. **型チェック**
   ```bash
   pnpm typecheck
   ```

4. **ビルド確認**
   ```bash
   pnpm build
   ```

5. **テスト実行**
   ```bash
   pnpm test
   ```

6. **結果サマリー表示**
   - 各ステップの成功/失敗を一覧表示
   - 失敗したステップがある場合は詳細を表示

## パラメータ

- `--quick`: フォーマット、リント、型チェックのみ実行（ビルド・テストをスキップ）

引数: $ARGUMENTS

## エラー時の対応

- 各ステップでエラーが発生した場合、詳細なエラーメッセージを表示
- 修正方法のヒントを提供
- 残りのステップも実行して全体像を把握できるようにする
