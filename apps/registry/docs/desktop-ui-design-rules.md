# Desktop UI デザインルール

## 3階層レイヤー構造

Desktop UIは以下の3階層で奥行きを表現する。全画面共通のルール。

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: サイドバー (--bg-app)                       │
│ ┌─────────────────────────────────────────────────┐ │
│ │ Layer 2: メインコンテンツ (--bg-main)            │ │
│ │  余白: 上8px 右8px 下8px 左0px                   │ │
│ │  角丸: rounded-xl                                │ │
│ │  ライトモード: shadow-main                       │ │
│ │ ┌───────────────────────────────────────────┐   │ │
│ │ │ Layer 3: カード (--bg-card)                │   │ │
│ │ │  角丸: rounded-xl                          │   │ │
│ │ │  ライトモード: shadow-card                 │   │ │
│ │ └───────────────────────────────────────────┘   │ │
│ │                                                  │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

### Layer 1: サイドバー（最下層）

- 背景: `--bg-app`（ダーク: `#0a0a0a` / ライト: `#e8eaed`）
- 全画面の背景色として機能
- サイドバーのナビゲーション、ワークスペース名を配置

### Layer 2: メインコンテンツ

- 背景: `--bg-main`（ダーク: `#111111` / ライト: `#f4f5f7`）
- サイドバーの上に載る構成（`rounded-xl`で四隅丸め）
- 余白: 上右下に8px、左は0px（サイドバーと接する）
- ライトモード時: `--shadow-main` で浮遊感を演出
- ダークモード時: shadowなし（ボーダーのみ）

### Layer 3: カード（最上層）

- 背景: `--bg-card`（ダーク: `#18181b` / ライト: `#ffffff`）
- メインコンテンツ内のセクション単位の要素
- `rounded-xl` + `border: 1px solid var(--border)`
- ライトモード時: `--shadow-card` で浮遊感
- ページ内に `p-6` の余白を設けてからカードを配置

## CSS変数（テーマ）

### 背景色

| 変数              | ダーク                   | ライト             | 用途                         |
| ----------------- | ------------------------ | ------------------ | ---------------------------- |
| `--bg-app`        | `#0a0a0a`                | `#e8eaed`          | サイドバー、アプリ全体の背景 |
| `--bg-main`       | `#111111`                | `#f4f5f7`          | メインコンテンツエリア       |
| `--bg-card`       | `#18181b`                | `#ffffff`          | カード、パネル               |
| `--bg-card-hover` | `rgba(255,255,255,0.04)` | `rgba(0,0,0,0.02)` | ホバー状態                   |
| `--bg-active`     | `rgba(255,255,255,0.08)` | `rgba(0,0,0,0.06)` | アクティブ状態               |
| `--bg-input`      | `rgba(255,255,255,0.03)` | `rgba(0,0,0,0.02)` | 入力フィールド               |

### シャドウ

| 変数            | ダーク | ライト                                                    | 用途                     |
| --------------- | ------ | --------------------------------------------------------- | ------------------------ |
| `--shadow-main` | `none` | `0 8px 32px rgba(0,0,0,0.08), 0 2px 8px rgba(0,0,0,0.04)` | メインコンテンツの浮遊感 |
| `--shadow-card` | `none` | `0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04)`  | カードの浮遊感           |

### テキスト

| 変数               | ダーク    | ライト    | 用途                           |
| ------------------ | --------- | --------- | ------------------------------ |
| `--text-primary`   | `#ffffff` | `#111827` | 見出し、強調テキスト           |
| `--text-secondary` | `#a1a1aa` | `#4b5563` | 本文テキスト                   |
| `--text-muted`     | `#71717a` | `#6b7280` | ラベル、補助テキスト           |
| `--text-subtle`    | `#52525b` | `#9ca3af` | サブテキスト、プレースホルダー |

### ボーダー

| 変数              | ダーク                   | ライト    | 用途         |
| ----------------- | ------------------------ | --------- | ------------ |
| `--border`        | `rgba(255,255,255,0.08)` | `#e5e7eb` | 標準ボーダー |
| `--border-subtle` | `rgba(255,255,255,0.03)` | `#f3f4f6` | 薄いボーダー |

### バッジ

| 変数                      | ダーク                              | ライト                             | 用途         |
| ------------------------- | ----------------------------------- | ---------------------------------- | ------------ |
| `--badge-success-bg/text` | `rgba(52,211,153,0.1)` / `#34d399`  | `rgba(16,185,129,0.1)` / `#059669` | 成功、稼働中 |
| `--badge-warn-bg/text`    | `rgba(251,191,36,0.1)` / `#fbbf24`  | `rgba(245,158,11,0.1)` / `#d97706` | 警告、遅延   |
| `--badge-error-bg/text`   | `rgba(248,113,113,0.1)` / `#f87171` | `rgba(239,68,68,0.1)` / `#dc2626`  | エラー、停止 |

## レイアウト構成

- **Header**: なし（サイドバーに統合）
- **StatusBar**: なし（サイドバーに統合）
- **サイドバー**: 収納可能（220px ↔ 56px）、左上にワークスペース名+ロゴ
- **テーマ切替**: サイドバー下部に配置（Sun/Moonアイコン）

## ロゴのテーマ対応

テーマに応じてロゴ画像を切り替える:

- ダークモード: `.webp`（白背景付きロゴ）
- ライトモード: `.svg`（黒ロゴ or カラーロゴ）
- GitHubなど白/黒のみのロゴは `_white.svg` / `_black.svg` で明示的に分離
