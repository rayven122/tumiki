## 📊 レポート形式

各週次レポートには以下の項目が含まれます：

### 1. 今週の成果
- 完了したタスク
- 実装した機能
- 解決した課題

### 2. 進行中のタスク
- 現在取り組んでいる作業
- 進捗状況
- 予定完了日

### 3. 課題と対策
- 発生した問題
- 対策案
- 必要なリソース

### 4. 来週の計画
- 優先タスク
- マイルストーン
- 必要な決定事項

### 5. メトリクス
- 開発速度
- バグ発生率
- テストカバレッジ

## 📝 レポート作成ガイドライン

### ファイル名規則

```
週次業務報告_YYYY年M月D日.md
```

- 年は4桁、月日は必要に応じて1-2桁
- 日付は週の最終営業日（通常は金曜日）

### テンプレート

```markdown
# 週次業務報告 - YYYY年M月D日

## 概要
今週の主な成果と進捗の要約

## 今週の成果
```
